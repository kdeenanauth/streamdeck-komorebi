Title Format can utilize variables with the syntax ${variable}

Variables available are:

  • monitorIndex
  • monitorIndexOneBased
  • workspaceIndex
  • workspaceIndexOneBased
  • workspaceName
  • numWorkspaces - number of used workspaces on the monitor
  • layout
  • abbrevLayout - abbreviated version of layout
  • containerIndex
  • containerIndexOneBased
  • numWindows - number of windows in a workspace
  • activeWindowTitle
  • activeWindowExe

Leaving above blank will do nothing on key down

Setting the above will lock this key to showing state information for the specified monitor.

Setting the above will lock this key to showing state information for the specified workspace
